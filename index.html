專題研究介紹 / INTRODUCTION

Taiwan Weather Forecast Project Document

1. 研究動機與目的

台灣地處亞熱帶，季風與颱風帶來劇烈降雨，近年極端氣候頻發。本專案由高中資訊科技課程出發，旨在透過 **HTML5 Canvas 動態物理引擎技術** 與 **JSON 即時數據解析**，將極端低溫與強降雨以具科技美感的「冷調、冰冷風格」進行動態呈現，打破傳統氣象播報的沈悶感，提升青年學子對於地學數據的敏感度。

2. 核心技術應用

系統以前端核心技術打造：**Canvas API** 用於繪製即時物理下落雨滴與衝擊地表濺射的粒子特效；**Tailwind CSS** 構築高冷系毛玻璃微縮介面；**JavaScript ES6+** 實作非同步數據模擬更新與控制台交互，並搭配響應式設計，確保在平板與行動裝置上皆能展現極佳的科技視覺感。

3. 研究結論與預期成果

本專案成功實現了將「枯燥的氣象代碼」轉換成「具備未來感的可視化儀表板」。經實驗測試，冷色系（#0ea5e9 與暗灰色調）能有效降低使用者閱讀高負載數據時的焦慮感，未來可進一步接入台灣中央氣象署的 OPEN API 即時數據庫，朝向全自動化即時天氣預警儀表板的方向拓展。

系統架構與實作邏輯 / SYSTEM ARCHITECTURE

How It Works - System Data Flow

我們將整個天氣監控專案拆解為四大核心層，這奠定了系統的冰冷美學基礎：

  • 01
    物理降雨渲染引擎 (Canvas Physics Engine)
    利用 requestAnimationFrame 驅動的高幀率雨水粒子動畫。
  • 02
    微縮數據調度器 (Data Dispatcher)
    模擬台灣各地氣候測站的反饋機制，並維持定時刷新的動態特徵。
  • 03
    冰冷交互介面層 (Frost UI Layer)
    基於冷灰、冰藍與半透明亞克力效果構成的無邊框儀表板。
